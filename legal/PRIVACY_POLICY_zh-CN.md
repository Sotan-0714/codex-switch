# Apivot 隐私政策

适用于 Apivot 1.0.1。

Apivot 是本地优先的桌面软件，不提供账号系统、广告、遥测、分析统计或 Hardcopia 云同步。

## 本地处理

Profile、API Key、Base URL、模型、Headers、配置路径、日志、备份、用量统计、聊天记录选择状态和许可接受记录默认保存在本机。

默认位置：

- Windows：`%APPDATA%\Apivot\`
- macOS：`~/Library/Application Support/Apivot/`

## 网络请求

只有在你刷新模型、测试连接、启动客户端或运行本地适配器时，程序才会向界面中显示的端点发送请求。

本地适配器仅监听 `127.0.0.1`，并根据你的本地 Profile 设置转发或转换请求。

## 用量统计

用量统计仅保存在本机，且只来自 Apivot 可观测的请求，例如本地适配器流量，以及返回 `usage` 的连接测试。如果服务商不返回 usage 字段，统计可能不完整；它不等同于官方服务商账单。

## 聊天记录

记录页只读取 Codex 和 Claude Code 的本机会话/历史文件，用于显示、筛选和选择性删除。删除聊天记录不会删除配置、登录状态或备份。

## 第三方服务

你选择的 API 服务商可能会根据其自身政策处理请求内容。Hardcopia 不会通过本软件收集或出售你的本地应用数据。
