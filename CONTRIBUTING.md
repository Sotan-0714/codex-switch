# Contributing

Thanks for your interest in Apivot.

## Development Setup

```bash
npm install
npm run build:renderer
npm start
```

## Pull Requests

- Keep changes focused and easy to review.
- Do not commit real API keys, provider URLs, local profile data, logs, backups, or release binaries.
- Update README, legal notes, or user-facing copy when behavior changes.
- Run the most relevant build or verification command before submitting.

## Useful Commands

```bash
npm run build:renderer
npm run legal:notices
npm run dist:win
```

macOS release packages must be built on macOS or GitHub Actions macOS runners.
